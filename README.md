# aws-webapp-deploy

## What this project does

This is a simple Node.js web application deployed on an AWS EC2 instance.
It returns a message in the browser to confirm the app is running.

---

## How to run manually

### 1. Install Node.js

```bash
sudo apt update
sudo apt install nodejs -y
```

### 2. Clone the repository

```bash
git clone https://github.com/zencodingllc/aws-webapp-deploy.git
cd aws-webapp-deploy
```

### 3. Run the app

```bash
node app.js
```

### 4. Open in browser

Go to:
http://3.141.22.95:3000
